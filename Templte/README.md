# SpamGuard AI 🛡️
An intelligent SMS/Email spam detection system built with **Python**, **Flask**, and **Scikit-Learn**.

## 🚀 Features
- **Machine Learning**: Uses Multinomial Naive Bayes for classification.
- **NLP**: Implements TF-IDF Vectorization and custom text preprocessing.
- **Visuals**: Generates a real-time Confusion Matrix and classification reports.
- **Responsive UI**: A modern "Glassmorphism" dashboard built with Bootstrap 5.

## 🛠️ Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/Spam-Detection-System.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Access at: `http://127.0.0.1:5000`

## 📊 Model Performance
- **Accuracy**: ~98%
- **Algorithm**: Multinomial Naive Bayes
- **Dataset**: UCI SMS Spam Collection
